> ⚠️ **Beta — interpret with caution.** The molecular dynamics task is preliminary. Both the CFPMD-26 reference test set and the metrics are still evolving and are actively being checked for correctness. Treat the current MD rankings and values as indicative only — they may still change as we validate the pipeline and grow the test set.
>
> The reference set currently holds 17 structures (the CFPMD-26 dataset spanning pure metals, alloys, high-entropy alloys, transition-metal dichalcogenides, perovskites and molecular crystals at 293–1500 K); an upcoming **v2 release will grow this AIMD test set to 100 structures**. All models currently on the leaderboard were run through a single unified script, [`models/run_md.py`](https://github.com/janosh/matbench-discovery/blob/main/models/run_md.py). If your model isn't listed yet, we invite you to run it and submit your metrics via a [pull request](https://github.com/janosh/matbench-discovery/blob/main/contributing.md).
