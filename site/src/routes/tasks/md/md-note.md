> ⚠️ **Beta — interpret with caution.** The molecular dynamics task is preliminary. Both the DynaMat v1.0 reference test set and the metrics are still evolving and are actively being checked for correctness. Treat the current MD rankings and values as indicative only — they may still change as we validate the pipeline and grow the test set.
>
> The reference set currently holds 17 structures (DynaMat v1.0, spanning pure metals, alloys, high-entropy alloys, transition-metal dichalcogenides, perovskites and molecular crystals at 293–1500 K); an upcoming **v2 release will grow this AIMD test set to ~100 structures**. Collabs to grow it even further welcome! The public reference data intentionally omits energies and forces. Energy/force RMSEs shown here are maintainer-computed private-label diagnostics and are excluded from CMDS, which ranks trajectory-level observables plus speed. All models currently on the leaderboard were run through a unified script, [`models/run_md.py`](https://github.com/janosh/matbench-discovery/blob/main/models/run_md.py). If your model isn't listed, we invite you to run it and submit your metrics via [PR](https://github.com/janosh/matbench-discovery/blob/HEAD/contributing.md).
