# AlchemBERT

We tested the IS2RE task of Matbench Discovery using BERT. The training-validation ratio was 9:1. When the learning rate was 10^-5, the training stopped approximately at 1000 epochs.

```txt
val_MAE=0.0582
test_MAE=0.111
```
